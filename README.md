# vueit

View + Model-View aspect of the Trakit application.
Technologies:
* Vue 2
* Vuex (coming soon)
* Axios - API calls
* Vue Router



## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
